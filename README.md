# Ratio
Ratio calculator

## Features
- Calculate ratios and resolutions
- Swap ratio orientation with a click
- Multiply or divide a resolution
- List of common ratio presets out of the box
- Save custom presets
- Export png image as templates
- Copy values to the clipboard
